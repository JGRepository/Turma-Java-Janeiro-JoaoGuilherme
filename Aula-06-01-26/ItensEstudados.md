## Itens Estudados

### Qual o método que inicia a execução no Java?
- O método `public static void main(String[] args)` é o ponto inicial onde o programa começa a rodar.

### O que é uma Classe?
- É uma estrutura que define como um objeto deve ser criado e como ele deve se comportar.

### Como criar uma Classe?
- Usando a palavra `class` seguida do nome da classe e seu bloco de código.

### O que é um Método?
- É um conjunto de instruções que executa uma tarefa específica dentro de uma classe.

### O que é uma variável?
- É um local da reservado da memória RAM usado para armazenar dados.

### O que é uma condicional (comando if)?
- É uma estrutura que permite executar um código apenas quando uma condição é verdadeira; para comparar `String`, use `equals()`.

### O que é Herança?
- É o mecanismo que permite que uma classe reutilize código de outra classe.

### Como comparar String?
- Usando o método `equals()` para comparar o conteúdo das strings, não a referência.

### Como receber dados do usuário pelo console?
- Utilizando a classe `Scanner` para ler informações digitadas no teclado.

### Por que tivemos que migrar do Java 1 para o Java 17?
- Para utilizar recursos modernos que não existiam no Java 1.

### O que é compilar?
- É o processo de transformar o código-fonte em bytecode que a JVM entende.

### O que é executar?
- É rodar o bytecode compilado na JVM para o programa funcionar.

### O que é erro de compilação?
- É um erro no código que impede o programa de ser compilado.

### O que é erro de execução?
- É um erro que acontece enquanto o programa está rodando.

### Como usar herança no Java?
- Criando uma classe que estende outra usando a palavra-chave `extends`.

### Como fazer concatenação de String?
- Unindo textos usando o operador `+` ou o método `concat()`.
